Periodical Sales Report

  • This module will print period wise sale report. Print the today , last week, months sales report

  • This module also allow to print report for particular date duration.

  • You can also print the report for particular state or for all state.

1. You have to just click on Sales -> Reports -> Periodical Sales Report

2. From popup you need to select either period or date duration and state.

3. Sample report.
